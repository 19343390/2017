# Automating GIS-processes - Sphinx + GitHub Pages

Source documents for maintaining the [Automating GIS-processes course pages](https://automating-gis-processes.github.io/2016/).

The docs are written in [Sphinx](http://www.sphinx-doc.org/en/1.4.9/) and all the rst files for the lesson contents are located in the [source](source/) -folder. Build html pages are located in the [docs](docs/) -folder.
